scikit-learn学习笔记

README

scikit-learn库同样是数据分析中重要的库之一，调用时简称sklearn。

使用sklearn库，可以根据DataFrame对象建立模型(Fitting model)，从而使用模型进行预测。

模型的建立是数据科学中的关键，有条件应该自己了解背后的算法并实现其功能，本文仅介绍scikit的使用。
